Project title

Ouija

Tagline

A student-safe AI experiment interpreter for middle and high school science labs.

Built with

React, Vite, TypeScript, Express, AI Architecture Map, UX and Accessibility Proof, OpenAI Responses API, Composio Sessions/MCP planning, Recharts, Vitest, Playwright, and Vercel.

Short description

Ouija helps students describe a lab, identify the experiment, inspect expected science patterns with citations, enter or paste their own table data, compare the graph against an expected overlay, and get hints without receiving a generated lab report.

Judging Map

Problem + relevance Middle/high school lab reasoning

Students have data and sources but often need help connecting variables, graphs, repeated trials, and claims.

AI design Hybrid, inspectable, guarded

Template matching, optional web-search grounding, evaluation checks, runtime proof, and source-audit handoffs stay visible.

UX + design Student mode first, proof visible

The default workflow prioritizes the lab, while UX Proof shows responsive layout, named controls, clickable citations, and integrity prompts.
